# Python Programming Minor Project Assignment

## Topic
Design a website for "Club Cafe".

## Requirement Gathering

### Essential Features

*   **Menu with Cake Varieties:**
    *   **Signature Cakes:** Classic Black Forest, Red Velvet Delight, Vanilla Buttercream, Pineapple Fresh Cream, Chocolate Truffle
    *   **Premium Designer Cakes:** Ferrero Rocher Cake, Oreo Crunch Cake, Blueberry Cheesecake, Butterscotch Caramel Cake
    *   **Cupcakes:** Chocolate Fudge Cupcake, Vanilla Sprinkle Cupcake, Red Velvet Cupcake, Oreo Cupcake
    *   **Pastries:** Chocolate Pastry, Black Forest Pastry, Strawberry Pastry, Butterscotch Pastry
*   Online ordering system
*   Gallery showcasing products
*   Contact form for inquiries
*   "About Us" page
*   Location & timings information
*   Reviews / testimonials section

## Technical Stack Suggestions (For Student Projects / Custom Tools)

*   **Frontend:** HTML5, CSS3, JavaScript
*   **Backend (optional):** PHP / Python (Django) / Node.js
*   **Database:** MySQL / MongoDB
